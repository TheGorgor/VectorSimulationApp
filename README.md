
  # Physics Simulation App UI

  This is a code bundle for Physics Simulation App UI. The original project is available at https://www.figma.com/design/JXOVIO5Mv1mv4TMSjwfr0G/Physics-Simulation-App-UI.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
                    
  Run vector-simulation.exe file to start the app.
